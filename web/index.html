Terragravity Hermes

Talk to your Hermes agent from anywhere. One command to install, with a desktop shortcut.

curl -fsSL https://terragravity.cloud/install.sh | sh

Paste into Terminal. Installs to ~/.terragravity, adds a tg command, and drops a shortcut on your Desktop.

curl -fsSL https://terragravity.cloud/install.sh | sh

Paste into your terminal. Adds a tg command and a launcher in your applications menu + Desktop.

irm https://terragravity.cloud/install.ps1 | iex

Paste into PowerShell. Adds a tg command to your PATH and a Desktop shortcut. No admin needed.

  1. Run the install command for your OS above.
  2. Add your API key — edit ~/.terragravity/config.env and set TG_API_KEY.
  3. Launch — double-click the desktop shortcut, or run tg chat in a terminal.
What it installs

A small, dependency-free Python launcher (tg) that connects to your Terragravity edge gateway over HTTPS. Commands: tg health, tg chat, tg agent <task>, tg status <job>, tg logs. Requires Python 3 (already on most systems).
